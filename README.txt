# PROJECT OVERVIEW

This repository contains the complete codebase for the Trust-Weighted Agentic IDS. Unlike traditional static intrusion detection systems, this framework leverages a dynamic Agentic AI architecture. It utilizes a centralized Orchestrator that routes network traffic to specialized Expert Agents and continuously adjusts their trust weights based on real-time Bayesian logic and active learning.

The system provides cross-domain robustness and was evaluated against multiple datasets, including:

* CIC IoMT 2024
* BoT-IoT
* ToN-IoT
* CIC-DDoS 2019
* CIC-IDS 2018

# SYSTEM ARCHITECTURE & FILE STRUCTURE

1. ORCHESTRATION & UI

---

orchestrator_master.py

* Central asynchronous Model Context Protocol (MCP) orchestrator.
* Evaluates an Autoencoder to filter benign traffic.
* Delegates threat analysis to expert agents.
* Calculates the final fused Bayesian probability.

Final_Parallel_Tester.py

* High-speed multiprocessing distributor.
* Assigns tasks to expert worker processes across separate CPU cores.
* Maximizes throughput and processing efficiency.

app.py

* Live Streamlit dashboard.
* Displays:

  * Rolling accuracy
  * Agent trust scores
  * Threat probabilities
  * Misclassification queue

---

2. EXPERT AGENTS

---

The framework deploys three specialized FastMCP server agents:

mqtt_agent.py

* Specialized in MQTT-based threats.
* Utilizes an XGBoost model.

volumetric_agent.py

* Specialized in Volumetric, DoS, and DDoS attacks.
* Utilizes a LightGBM model.

recon_agent.py

* Specialized in scanning and probing activity.
* Utilizes a Random Forest model.
* Supports active learning trust adjustments.

---

3. ACTIVE LEARNING & RETRAINING

---

retrain_models.py

* Reads the master_retrain_queue.csv generated by the orchestrator.
* Incrementally retrains:

  * LightGBM
  * XGBoost
  * Random Forest
* Uses misclassified flows for active learning.
* Safely backs up original models to the models_backup/ directory.

# HOW TO RUN THE SYSTEM

1. LIVE INTERACTIVE DASHBOARD

---

To view the live execution of the Agentic IDS with trust adaptation, run both the orchestrator and dashboard simultaneously.

Terminal 1 - Start the Orchestrator

```
python3 orchestrator_master.py
```

Notes:

* Evaluates datasets in real time.
* Outputs performance metrics.
* Queues misclassified flows into:
  master_retrain_queue.csv

Terminal 2 - Start the Dashboard

```
streamlit run app.py
```

---

2. HIGH-SPEED PARALLEL BENCHMARK

---

To test maximum system throughput using CPU multiprocessing:

```
python3 Final_Parallel_Tester.py
```

Optional benchmark execution:

```
python3 Benchmark.py
```

This generates throughput and stability graphs suitable for presentations and analysis.

---

3. ACTIVE LEARNING RETRAINING

---

After the orchestrator populates the misclassification queue:

```
python3 retrain_models.py
```

This process performs incremental learning by adding boosting rounds and estimators to the expert models.

Output includes:

* Before retraining accuracy
* After retraining accuracy
* Updated model artifacts
* Backup copies of original models
